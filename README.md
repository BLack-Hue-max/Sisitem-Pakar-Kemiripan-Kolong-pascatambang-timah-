# Sisitem-Pakar-Kemiripan-Kolong-pascatambang-timah-
Sistem pakar kemiripan kolong pascatambang timah adalah sistem yang memberikan kemungkinan persentase kemiripan dari kolong yang belum diketahui kualitas airnya.
Untuk melihat dataset dan program dapat melihat pada Kaggel dengan kata kunci kolong pascatambang timah.
