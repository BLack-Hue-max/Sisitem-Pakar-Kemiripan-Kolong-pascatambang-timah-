# Sisitem-Pakar-Kemiripan-Kolong-pascatambang-timah-
Sistem pakar kemiripan kolong pascatambang timah adalah sistem yang memberikan kemungkinan persentase kemiripan dari kolong yang belum diketahui kualitas airnya.
Untuk melihat dataset dan program dapat melihat pada Kaggel dengan kata kunci kolong pascatambang timah.


Program pembuatan arsitektur CNN dan KNN dengan nama Kolong_CNN_Skripsi dijalankan di Colab
Program pengujian CNN untuk kemiripan kolong dengan nama CNN only dijalankan di Colab
Program pengujuan KNn untuk kemiripan kolong dengan nama KNN only dijalankan di colab
Hasil pemodelan CNN dengan format file .h5
Hasil pemodelan KNN dengan format .pkl serta mapping warna
Program menjalankan Kemiripan kolong menggunakan FLask terdiri dari dijalakan di lokal komputer dengan anaconda
-Dataset.csv
-CNN.h5
-KNN.pkl
-mapping warna.pkl
-citra kolong
